Request Header Parser API

This is a simple API that returns some of the information included in the HTTP request header sent through by your browser. It has been teseted using Safari, mobile Safari, Chrome and Firefox on Mac OS X. Support for Opera, Edge, and Internet Explorer and other systems were added but have not been fully tested.

API Call:

http://honmanyau-header-parser-api.glitch.me/whoami

Example Output:

{"ipaddress":"100.100.100.10","language":"en-US","system":"Macintosh; Intel Mac OS X 10.12","browser":"Firefox","browserversion":"53.0"}
